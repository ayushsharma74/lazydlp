# lazydlp
A minimal TUI downloader for YouTube (and 1700+ other sites), powered by **yt-dlp** and **Bubble Tea**.

## Installation

### macOS & Linux (Homebrew)

```bash
brew tap ayushsharma74/tap
brew install lazydlp
```

### Arch Linux (AUR)

```bash

yay -S lazydlp-bin


```

### Build From Source (Go)

```bash
go install github.com/ayushsharma74/lazydlp/cmd/lazydlp@latest

```
## Usage

Simply run the command and follow the prompts:

```bash
lazydlp

```
![output](https://github.com/user-attachments/assets/e58524f0-a121-4c16-adf6-d766d9b183bb)

### Requirements

* [yt-dlp](https://github.com/yt-dlp/yt-dlp) (Required)
* [ffmpeg](https://ffmpeg.org/) (Recommended for high-quality merging)
